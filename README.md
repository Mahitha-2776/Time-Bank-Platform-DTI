# ⏰ TimeBank — Exchange Skills, Build Community

A community-driven skill exchange platform where people trade time instead of money. Offer your expertise, request help from others, and earn **Time Credits** for every hour you contribute.

## ✨ Features

- 🔍 **Search** — Find services, skills, or members instantly with debounced search
- 🗂️ **Category Filtering** — Browse by Education, Home, Tech, Creative, Wellness, and more
- 🔄 **Offer / Request Toggle** — Switch between skills offered and skills needed
- 📊 **Sort Options** — Sort by Newest, Oldest, Time Credits, or Highest Rated
- 🔖 **Bookmarks** — Save listings for later with a single click
- 📝 **Post a Listing** — Share your skill or post a help request with the community
- ⭐ **Ratings & Reviews** — See community feedback and top contributors
- 📈 **Time Balance Tracker** — View your earned and spent time credits
- 🔔 **Notifications** — Stay updated on activity and requests
- 👤 **User Profile Dropdown** — Access profile, saved listings, and settings
- 🌙 **Dark Mode** — Full light/dark theme toggle with persistent preference
- 📱 **Responsive Design** — Works seamlessly on desktop and mobile

## 🚀 Getting Started

No installation or build tools needed — this is a plain HTML/CSS/JS project.

**Option 1 — Open directly:**
1. Download or clone this repository
2. Open `index.html` in your browser

**Option 2 — Clone via Git:**
```bash
git clone https://mahitha-2776.github.io/Time-Bank-Platform-DTI/
cd timebank-app
# Open index.html in your browser
```
